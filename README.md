# Портфолио Ильи Шиятого

Статический сайт-портфолио в sci-fi/IronShift стилистике. Проект рассчитан на учебное портфолио для колледжа и как личная страница, которую можно постепенно развивать: добавлять реальные проекты, демо, ссылки на репозитории, сертификаты и материалы.

## Технологии

- HTML
- CSS
- JavaScript

## Запуск

Откройте `index.html` в браузере.

Основная версия сайта находится в `index.html`.

## Что можно развивать дальше

- Добавить реальные проекты, демо и ссылки на GitHub, когда они появятся.
- Подписать сертификаты конкретными названиями, датами и результатами.
- Оптимизировать изображения и сделать отдельные превью для галереи.
- Улучшить доступность модального просмотра изображений.
