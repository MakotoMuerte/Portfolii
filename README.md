# Портфолио

Статический сайт-портфолио в sci-fi/IronShift стилистике. Проект рассчитан на учебное портфолио для колледжа и как личная страница, которую можно постепенно развивать: добавлять реальные проекты, демо, ссылки на репозитории, сертификаты и материалы.

## Технологии

- HTML
- CSS
- JavaScript

## Запуск

Откройте `index.html` в браузере.

Основная версия сайта находится в `index.html`.
